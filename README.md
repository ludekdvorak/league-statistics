# league-statistics
Repository for league of legends statistics using React and java.
User loads his web where he writes his league ID and loads his statistics
App is just for personal use only

## Functionality
- Loading your personal profile
- Calculating matches from match history
- Seeing statistics for further improving performance

## Used Technologies
- React
- Java
- SpringBoot
- Git, GitHub

## How to use
1. Clone repository
2. Load npm servers for both front&backend servers
